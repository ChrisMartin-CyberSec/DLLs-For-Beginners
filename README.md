# DLLs-For-Beginners

## Simple DLL linking using C++ executable


This repository contains two files: `main.cpp` and `dlltest.cpp`


The `main.cpp` will be compiled as the **executable** .


The `dlltest.cpp` will be compiled as the **dll**.
